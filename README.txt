== Description ==

Solutions Ad Manager is a simple ad manager that allows you to display advertisements in the form of pictures, YouTube videos, and many more embeddable elements in to a widget. This plugin comes with builtin analytics to track how many clicks you get for each ad.

== Installation ==

1. Extract `solutions-ad-manager.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Display your ads by editing your widgets

== Frequently Asked Questions ==

= How to use Solutions Ad Manager =
Create an Ad
Create a Group
Add your ads to the Group
Select Widgets under Appearance
Choose the widget you would like to display
How to show titles and controls on YouTube videos
Select Options.
Choose the options you would like under "YouTube Options".
Press Save

= How to show titles and controls on YouTube videos =
Select Options.
Choose the options you would like under "YouTube Options".
Press Save

= How to backup your ads =
Ads can be backed up by using the Export feature that comes with WordPress. And if you need to restore them use the Import feature that comes with WordPress.
